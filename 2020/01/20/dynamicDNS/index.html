pi-hole、Domoticz、Homebridge都已经安装好了，是时候设置个域名了，考虑到Cloudflare的强大，此次的DDNS将使用Cloudflare来实现。

Cloudflare 控制台配置

1️⃣ 登录cloudflare控制台。

2️⃣ 添加你要进行解析的域名

3️⃣ 添加一条默认的A记录，指向一个ipv4的地址。

Global API Key

如果使用动态域名，需要使用全局apikey来登录更新A记录的值

  1. 在CloudFlare管理后台登录个人面板
  2. 向下滑动找到全局api
  3. 记录下api后边使用
